## 🎯 項目核心理念

### 設計理念
```
讓使用者「自然而然地想要記錄工作」
```

### 核心價值
- **零門檻**：打開就能記錄
- **即時滿足**：完成任務立刻有成就感
- **智能引導**：系統主動提醒
- **習慣養成**：連續記錄可視化
- **愉悅體驗**：流暢動畫 + 美觀介面

```

├── components/          # 組件庫

```

## 🎨 設計系統

### 配色
```css
主色：#6366f1 (Indigo)
輔色：#8b5cf6 (Purple)
成功：#10b981 (Green)
警告：#f59e0b (Orange)
背景：#f9fafb (Gray)
```

### 設計原則
1. **行動驅動優先** - 降低記錄門檻
2. **一致的視覺語言** - 統一設計系統
3. **流暢的互動體驗** - 即時反饋
4. **漸進增強** - 先基礎後高級
